# 寒酸啟示錄廣告小遊戲（BROKEOUT SURVIVAL）

一款以《寒霜啟示錄》(*Whiteout Survival*) 冰原末日題材為靈感的
**單檔 HTML5 Canvas 2D 等距 (isometric) 放置經營廣告小遊戲**。

> 遊戲內採用諧音惡搞品牌「**寒酸啟示錄 / BROKEOUT SURVIVAL**」。
> 本作是一個**可遊玩的廣告創意 / 獨立 web 小遊戲**，**並非**它所致敬的 SLG 本體。
> 與正版的關係請見 [聲明.md](聲明.md)。

## 初衷

主要是因為**朋友想玩這種類型的小遊戲**，就動手做了一個。

順帶想驗證一件事：**靠 vibe coding 就能做到這樣的成果**——單檔、無框架、無建置，
邊寫邊玩邊調。但前提是，**動手開發之前要先把規格弄好**：先有
[`GAMEPLAY_SPEC.md`](GAMEPLAY_SPEC.md) 與 [`經濟與單位定義.md`](經濟與單位定義.md)
把玩法、數值與經濟曲線想清楚，後面的實作才跑得順、改得動。

## 核心玩法

採集 → 搬運 → 交付解鎖 → 加工 → 販售 → 升級 → 擴張

走到採集點自動採「單一素材」堆在背上，搬去三種去向：

- **交付解鎖**：把對的素材交進建造框（種類錯不收），填滿即解鎖新站點/區域。
- **加工**：生牛 →屠宰台→ 牛肉；木材＋牛肉 →烤爐→ 口糧（高價成品）。
- **販售**：把成品交給 FIFO 排隊顧客 → 掉現金 → 玩家走過去撿錢入帳。

賺到的錢拿去 7 座升級台（站上去自動扣款，含招工），招募伐木工 / 牧場工 /
店員自動化整條產線，產量滾雪球式放大。為廣告節奏設計，1–3 分鐘可體驗完整循環，
無死亡、無戰鬥。

## 執行方式

**直接用瀏覽器開啟 `index.html` 即可**——無需安裝、無建置步驟、無相依套件。

- 雙擊 `index.html`（`file://`）或用任意靜態伺服器，例如：
  ```
  python3 -m http.server
  # 然後瀏覽 http://localhost:8000/index.html
  ```
- 為**直向 (~9:16)** 設計，桌機與觸控皆可；桌機會置中加邊。
- 進度自動存於瀏覽器 `localStorage`（每 ~2 秒、切換分頁時），重開續玩；
  右上角「↺ 重置」可清檔重來。

## 技術

- 純 **HTML + CSS + 原生 JavaScript**，全部在一個 `index.html`（約 480 行）內。
- 等距 Canvas 2D 渲染、固定步進主迴圈 (`requestAnimationFrame` + 累加器,
  `STEP=1/60`)、`localStorage` 存檔。
- **無框架、無 build、無 npm、無測試**。「測試」＝載入頁面實際試玩。
